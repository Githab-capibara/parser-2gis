## Известные ограничения

### Аутентификация

Проект не хранит и не требует аутентификации пользователей 2GIS. Используйте ответственно.

### rate limiting

Слишком частые запросы могут привести к блокировке IP. Настройте `rate_limit` в конфигурации.

### Данные

- Не сохраняйте персональные данные без необходимости
- Соблюдайте условия использования 2GIS

### Зависимости

Регулярно обновляйте зависимости:

```bash
pip audit
```

### Конфиденциальность
