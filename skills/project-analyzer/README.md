# Project Analyzer Skill

Скилл для QWEN CLI для комплексного анализа проектов, исправления ошибок, оптимизации кода и синхронизации с GitHub.

## Возможности

- **Анализ структуры проекта** — автоматическое сканирование всех файлов проекта
- **Поиск ошибок** — логические, синтаксические ошибки и уязвимости безопасности
- **Оптимизация кода** — улучшение производительности и читаемости
- **Перевод комментариев** — все комментарии переводятся на русский язык
- **Синхронизация с GitHub** — автоматический коммит и пуш изменений

## Установка

Скилл находится в директории:
```
/home/d/Qwen/skills/project-analyzer/
```

QWEN CLI автоматически обнаружит скилл при запуске.

## Использование

### Базовое использование

Просто попросите QWEN проанализировать проект:

```
Проанализируй весь проект и исправь ошибки
```

```
Проверь код в main.py и переведи комментарии на русский
```

```
Оптимизируй этот код и синхронизируй с GitHub
```

### Команды скриптов

#### timeout_command.py
Выполнение команд с таймаутом 5 минут:
```bash
python scripts/timeout_command.py "git status" --timeout 300000
```

#### github_sync.py
Синхронизация с GitHub:
```bash
# Полная синхронизация
python scripts/github_sync.py sync -m "fix: исправление ошибки"

# Только статус
python scripts/github_sync.py status

# Только коммит
python scripts/github_sync.py commit -m "refactor: оптимизация кода"
```

#### code_analyzer.py
Статический анализ кода:
```bash
# Анализ одного файла
python scripts/code_analyzer.py main.py

# Рекурсивный анализ директории
python scripts/code_analyzer.py /path/to/project --recursive

# Вывод в JSON формате
python scripts/code_analyzer.py . --format json
```

## Правила

### Таймауты
**НА КАЖДУЮ КОМАНДУ** устанавливается таймаут 5 минут (300000 мс).

### Язык комментариев
Все комментарии в коде должны быть на **русском языке**. Технические термины (имена функций, классов, переменных) остаются на английском.

### GitHub
- Все изменения синхронизируются с репозиторием
- Файл QWEN.md никогда не включается в коммиты
- Используются информативные сообщения коммитов

## Примеры сообщений коммитов

- `fix: исправление ошибки деления на ноль`
- `refactor: выделение повторяющейся логики в функцию`
- `comments: перевод комментариев на русский язык`
- `optimize: улучшение производительности обработки данных`
