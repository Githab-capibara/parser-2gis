## 🎯 BREAKING CHANGES

### Минимальные breaking changes:

1. **`main.py._validate_url()`**  
   Теперь возвращает `tuple[bool, str | None]` вместо `bool`

2. **`config.py.Configuration.merge_with()`**  
   Добавлен параметр `max_depth: int = 50`

3. **`validator.py.validate_email()`**  
   Добавлен параметр `check_mx: bool = False`

4. **`cache.py.clear_batch()`**  
   Выбрасывает `ValueError` при некорректном хеше
