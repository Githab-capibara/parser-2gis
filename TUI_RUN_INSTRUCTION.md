# Инструкция по запуску Parser2GIS с новым TUI интерфейсом

## Быстрый старт

### Запуск парсинга Омска (10 потоков, 93 категории)

```bash
./run.sh
```

Это запустит:
- ✅ Новый TUI интерфейс на pytermgui
- 📊 Автоматический парсинг города Омск
- 📂 Все 93 категории
- 🔧 10 параллельных браузеров
- 🎨 Современный интерфейс с прогресс-барами и логами

---

## Режимы работы

### 1. Парсинг Омска с TUI (по умолчанию)
```bash
./run.sh
```

### 2. Запуск TUI без автоматического парсинга
```bash
./run.sh --tui
```

### 3. Запуск нового TUI вручную
```bash
./run.sh --tui-new
```

### 4. Запуск с аргументами (CLI режим)
```bash
./run.sh --cities omsk spb --categories-mode --parallel-workers 10
```

## Возможности нового TUI (pytermgui)

### Главное меню
- 🚀 Запустить парсинг
- 📁 Выбрать города (204 города)
- 📂 Выбрать категории (93 категории)
- ⚙️ Настройки браузера
- 🔧 Настройки парсера
- 📊 Настройки вывода
- 📈 Просмотр кэша
- ℹ️ О программе

### Экран парсинга
- Прогресс-бары: URL, страницы, записи
- Статистика в реальном времени
- Логи с цветовой индикацией
- Кнопки: Пауза, Стоп, Свернуть

### Настройки
- **Браузер**: headless, память, путь к Chrome
- **Парсер**: retry, лимиты, GC
- **Вывод**: формат, кодировка, опции CSV

## Конфигурация парсинга Омска

При запуске через `./run.sh` используются следующие настройки:

### Город
- **Код**: omsk
- **Название**: Омск

### Категории
- **Количество**: 93 (все доступные)
- **Примеры**: Кафе, Рестораны, Аптеки, Супермаркеты, и т.д.

### Потоки
- **Количество**: 10 параллельных браузеров

### Chrome
- **Headless**: Да (фоновый режим)
- **Без изображений**: Да (экономия трафика)
- **Тихий режим**: Да
- **Лимит памяти**: 2048 МБ

### Парсер
- **Stop on 404**: Да
- **Max consecutive empty pages**: 5
- **Max retries**: 3
- **Retry on network errors**: Да

### Вывод
- **Формат**: CSV (по умолчанию)
- **Кодировка**: utf-8-sig
- **Удаление дубликатов**: Да
- **Удаление пустых колонок**: Да

## Навигация в TUI

- **Tab / Shift+Tab** - переключение между элементами
- **Enter** - активация (кнопки, чекбоксы)
- **Esc** - назад / отмена
- **Ctrl+C** - выход
- **Мышь** - поддержка кликов

## Устранение проблем

### Ошибка "Виртуальное окружение не найдено"
```bash
./run.sh  # Автоматически создаст venv
```

### Ошибка импорта pytermgui
```bash
source .venv/bin/activate
pip install pytermgui
```

### Ошибка "Город Омск не найден"
Проверьте наличие файла `parser_2gis/data/cities.json`
